Voith API Challenge

This is the documentation page on how to use the API

GET /places, /places/all

This returns a list of all places

GET /place/:id

This returns a specific place with id specified in the path

Request Parameter: id

Example: /place/25

Response: 200 OK

GET /places/search

This this searches for items with the query in the request. The search is performed on the name and description fields of the data items

Query Parameter: searchText

Example: /places/search?searchText="Heidenheim"

Response: 200 OK
